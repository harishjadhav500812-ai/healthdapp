================================================================================
                              TROUBLESHOOTING
================================================================================

ISSUE: Backend won't start
---------------------------
FIX:
  - Run: npx kill-port 5000
  - Then run: cd backend && npm run dev
  - Check if MongoDB connection works


ISSUE: Frontend won't start
----------------------------
FIX:
  - Run: npx kill-port 5173
  - Then run: cd frontend && npm run dev
  - Make sure node_modules folder exists


ISSUE: Still see "Access Denied"
---------------------------------
FIX:
  - Check backend window shows "Server running on port: 5000"
  - Check frontend window shows "Local: http://localhost:5173/"
  - Both must be running at the same time!
  - Refresh browser (Ctrl+Shift+R)


ISSUE: Port already in use
---------------------------
FIX:
  - Backend: npx kill-port 5000
  - Frontend: npx kill-port 5173
  - Then start servers again
